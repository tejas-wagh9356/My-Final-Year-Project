# 🎙️ AI-Powered Voice Assistant with Vulnerability Scanner and Phishing URL Detection

## 📖 Description

This project is an advanced AI-based voice assistant built using Python.
It can execute voice commands and also enhance cybersecurity by detecting vulnerabilities and identifying phishing URLs.

## 🚀 Features

* 🎤 Voice Recognition (Speech-to-Text)
* 🔊 Text-to-Speech Response
* 🌐 Open websites and search queries
* 🛡️ Vulnerability scanning (basic security checks)
* 🔗 Phishing URL detection system
* 🗂️ Database support using SQL
* ⏰ Time, date, and system operations

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* SQL (Database)
* OOP Concepts
* URL/Network Security Libraries

## ⚙️ Installation

1. Clone the repository
2. Install dependencies:

   ```bash
   pip install speechrecognition pyttsx3 pyaudio requests
   ```
3. Run the main Python file

## ▶️ Usage

* Run the assistant
* Speak commands like:

  * "Open Google"
  * "Scan this website"
  * "Check if this URL is safe"
  * "What is the time?"
